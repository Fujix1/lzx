UnLZX for MDX / PDX
===================

X68000 MDX / PDX 向けの LZX 圧縮ファイルを解凍する Windows 用ツールです。

使い方
------

1. mdx_unlzx.exe を起動します。
2. ウィンドウに .mdx / .pdx ファイル、またはフォルダをドラッグ&ドロップします。
3. LZX 圧縮されているファイルだけが一覧に表示されます。
4. [解凍] を押すと、元ファイルを .bak に退避して、解凍済みファイルを元の名前で保存します。

注意
----

- 既に .bak がある場合は .bak1, .bak2 ... のように退避します。
- 保存時は一度 .tmp に書き出してから置き換えます。
- ファイル処理はローカルPC内で完結します。
